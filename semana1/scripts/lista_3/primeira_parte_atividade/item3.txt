3. Como você pode lidar com exceções em Java? Quais são as palavras-chave e as práticas comuns para o tratamento de exceções?

Em Java, você pode lidar com exceções usando as palavras-chave try, catch, finally e throw. Algumas práticas comuns incluem:

Bloco try-catch: Utilizado para envolver o código que pode gerar uma exceção, permitindo a captura e tratamento adequado.


try {
    // Código que pode gerar uma exceção
} catch (TipoDeExcecao e) {
    // Tratamento da exceção
}


Bloco finally: Utilizado para definir um bloco de código que será executado, independentemente de ocorrer uma exceção ou não.

try {
    // Código que pode gerar uma exceção
} catch (TipoDeExcecao e) {
    // Tratamento da exceção
} finally {
    // Código a ser executado sempre
}


Lançamento de Exceções (throw): Utilizado para explicitamente lançar uma exceção em determinada situação.

if (condicao) {
    throw new MinhaExcecao("Mensagem de erro");
}
